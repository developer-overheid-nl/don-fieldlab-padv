## Wat zijn de minimale eisen?

Om een Open Source project bruikbaar te maken heb je in ieder geval een aantal dingen nodig:

- Een readme `(README.md)`
- Een gedragscode `(CONTRIBUTING.md)`
- Een licentie `(LICENSE)`
- Een `publiccode.yml`
